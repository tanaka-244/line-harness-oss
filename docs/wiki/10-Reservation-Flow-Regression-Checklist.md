# 予約問診 回帰チェック

最終更新: 2026-04-18

このチェックは、予約問診フローが他の修正で壊れていないかを確認するための最低限の確認項目です。

## 対象

- `予約する`
- `予約・ご相談`
- 初診
- 再診
- メッセージで相談する
- 初診PDF生成と院側通知

## 事前条件

- Worker が最新デプロイ済みであること
- `CLINIC_LINE_USER_ID` が設定済みであること
- `PDF_BUCKET` に PDF 用アセットが存在すること
- 確認用の LINE アカウントで実機操作できること

## チェック項目

1. `予約する` を送る
- 最初の Flex が表示される
- 選択肢が `初めての方（初診） / 2回目以降（再診） / メッセージで相談する` になっている
- アンケート通知が重複発火しない

2. `予約・ご相談` を送る
- `予約する` と同じ最初の Flex が表示される

3. 初診を選ぶ
- 次の Flex が表示される
- 選択肢が `ケガの施術（保険） / 慢性症状（自費） / 美容鍼・美療メディセル` になっている

4. 初診 `ケガの施術（保険）`
- 質問が最後まで進む
- 最後に保険用の確認事項が表示される
- `同意する` を押して完了できる
- 院側 LINE に PDF 通知が届く
- PDF URL を開ける

5. 初診 `慢性症状（自費）`
- `症状が悪化するのはどんな時ですか？` が出ない
- `ご希望の施術はありますか？` が `鍼灸 / メディセル / 整体 / 相談して決める` の 4 択になっている
- 最後に慢性用の確認事項が表示される
- 完了後、院側 LINE に PDF 通知が届く

6. 初診 `美容鍼・美療メディセル`
- 次の選択肢が `美容鍼 / 美療メディセル（女性限定）` になっている
- `ご希望のメニューを教えてください` の質問が出ない

7. 初診 `美容鍼`
- 最後に美容鍼用の確認事項が表示される
- 完了後、院側 LINE に PDF 通知が届く

8. 初診 `美療メディセル（女性限定）`
- 禁忌分岐が動く
- 完了後、院側 LINE に PDF 通知が届く
- 人体図が下枠にかからない

9. 再診を選ぶ
- `今回のご予約内容を教えてください` が表示される
- 選択肢が `ケガ・急な痛み / 慢性症状・不調 / 美容鍼 / 美容メディセル（女性限定）` になっている
- 完了後、院側 LINE に再診通知が届く

10. メッセージで相談する
- `ご相談内容を自由にご記入ください` が表示される
- 1 通入力すると完了メッセージが返る
- 院側 LINE に相談通知が届く

## 壊れたときの戻し先

- 安定ブランチ: `codex/reservation-flow-restore`
- 安定タグ: `reservation-flow-stable-2026-04-18`

## 補足

- 予約問診に触る修正を入れたら、このチェックを最低 1 回通す
- 大きい変更前には branch と tag を残す
